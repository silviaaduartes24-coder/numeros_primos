Verificador de Números Primos

Una aplicación web elegante y minimalista diseñada para identificar números primos, con un enfoque en la estética artesanal y una experiencia de usuario fluida.

🎨 Características de Diseño

Paleta de Colores: Basada en tonos beige y café para transmitir calidez y profesionalismo.

Tipografía: Combinación de Inter (Sans-serif) para elementos funcionales y Lora (Serif) para una lectura elegante y legible.

Fondo Dinámico: Patrón constante de líneas fluidas animadas mediante CSS que brindan profundidad sin distraer al usuario.

Responsivo: Diseño adaptable a dispositivos móviles, tablets y computadoras de escritorio.

🛠️ Tecnologías Utilizadas

HTML5: Estructura semántica del aplicativo.

Tailwind CSS: Framework utilizado para el diseño rápido y moderno.

JavaScript (Vanilla): Lógica de cálculo optimizada para la verificación de números primos.

Google Fonts: Integración de fuentes personalizadas.

🧮 Lógica Matemática

El verificador utiliza un algoritmo optimizado que reduce la complejidad computacional:

Descarta números menores o iguales a 1.

Maneja excepciones rápidas para 2 y 3.

Evalúa divisibilidad utilizando la forma 6k ± 1, verificando solo hasta la raíz cuadrada del número ingresado.

✒️ Créditos

Este proyecto fue desarrollado por Silvia Duarte, asistida por IA, buscando un equilibrio entre funcionalidad técnica y diseño visual sofisticado.

Este proyecto reside en un único archivo primos.html para facilitar su portabilidad y ejecución inmediata en cualquier navegador.
